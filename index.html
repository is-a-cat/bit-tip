Project BIT-TIP

Bitcoin virtual tip jar

This project is in beta.

Bit-Tip is a new virtual tip jar. It is a jar-themed donation button that fills up with coins as your account fills up with bitcoins.


Your tip jar filling up.

Set 'Max' to the total amount you want the jar to hold and DON'T FORGET TO CHANGE THE ADDRESS TO YOUR OWN

This is an alternate code for forum signitures. It is a dynamic image that shows your balance as a tip jar. Clicking it will take you to a personalised tip button. Don't forget to fill out all of the details!
